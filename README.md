# 🏏 IPL Cricket Data Analysis (2008 - 2022)

## About
A data analysis project exploring 14 years of IPL cricket data
using Python. Built as part of my Data Science degree at SLIIT.

## Dataset
- 950 matches (2008 - 2022)
- 225,954 ball-by-ball records

## Key Findings
- 🏆 Mumbai Indians have the most wins in IPL history
- 🎯 Winning the toss does NOT guarantee winning the match
- 🔥 Virat Kohli is the all-time top run scorer in IPL
- 📈 IPL has grown significantly in matches per season over 14 years
